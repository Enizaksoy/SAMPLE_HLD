1. Introduction & Network Overview

1.1 Purpose and Scope

This document provides a comprehensive High Level Design (HLD) of the Core Packet Network (CPN). It covers the current-state architecture based on analysis of all 12 production router configurations across two sites, and serves as the reference baseline for the planned Cisco → Juniper MX migration.

The network provides mobile core connectivity (2G/3G/4G), enterprise services, internet access, roaming, and signaling transport for the mobile operator.

5.5 Bridge-Domain + BVI Architecture

The BVI (Bridge-Group Virtual Interface) provides the L3 gateway for each bridge domain. The architecture is:

  1. Physical/satellite ports connect to a bridge-domain (Layer 2 switching)
  2. The bridge-domain has a BVI interface (IRB) providing the L3 gateway IP
  3. The BVI is placed in a VRF (e.g., SIGTRAN, USERPLANE, 4G-S1-CP)
  4. The VRF redistributes into MPLS L3VPN (BGP VPNv4) for inter-site reachability
  5. Between PE90KKY03 and PE90KKY04, bridge-domains are extended over BE5042 (40G L2 trunk)
  6. HSRP runs on all BVI pairs for gateway redundancy
